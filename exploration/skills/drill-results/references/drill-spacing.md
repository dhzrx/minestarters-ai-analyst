# Drill Spacing Guidelines

## Recommended Spacing by Commodity and Classification

| Commodity | Measured | Indicated | Inferred |
|-----------|----------|-----------|----------|
| Gold (vein) | 12–25m | 25–50m | 50–100m |
| Gold (disseminated) | 25–50m | 50–100m | 100–200m |
| Copper porphyry | 50–75m | 75–150m | 150–300m |
| Copper VMS | 25–50m | 50–100m | 100–200m |
| Lithium (hard rock) | 25–50m | 50–100m | 100–200m |
| Lithium (brine) | 500–1000m | 1000–2000m | >2000m |
| Iron ore (BIF) | 50–100m | 100–200m | 200–400m |
| Nickel laterite | 50–100m | 100–200m | 200–400m |
| REE | 25–50m | 50–100m | 100–200m |

## Notes

- Drill spacing depends heavily on geological continuity and grade variability
- These are guidelines only — site-specific geostatistical analysis should determine actual spacing
- Variogram ranges provide the best guide to appropriate drill spacing
- Consider both along-strike and down-dip spacing
- Closer spacing required for higher-grade, more variable deposits
